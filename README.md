# Stress Free Flow Website

Simple React landing page for App Store traffic and Google indexing.

## Preview locally

From this folder:

```bash
python3 -m http.server 4173
```

Then open:

`http://localhost:4173`

## Notes

- This is a lightweight React site with no build step.
